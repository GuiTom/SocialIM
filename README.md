## 模块划分
1. 主页
    - 游戏
    - 语音房
2. 登录注册
    - 短信注册
    - 密码登录
    - 短信登录
    - 密码修改 
3. 发现(编辑)
    - 动态列表
    - 发布动态
4. 消息
    - 私聊
    - 群聊
5. 通讯录
    - 好友
    - 粉丝
    - 关注
6. 我的(个人中心)
    - 设置
      - 换绑手机
      - 黑名单
      - 修改密码
      - 注销账号
    - 展示和编辑个人资料
      - 头像
      - 昵称
      - 用户ID
      - 性别
      - 生日
      - 手机号
      - 用户等级
      - 财富等级
      - 所在地区
      - 个性签名
    - 客服和帮助
    - 版本检查
    - 退出登录
